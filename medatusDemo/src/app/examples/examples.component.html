General info

Medatus Buttons are native <button> or <a> elements enhanced with Medatus Design styling.


Medatus Buttons Group are native <div> elements enhanced with the appearance of a buttons set from Medatus Design styling.

Theming

Medatus Buttons can be colored in terms of the current theme using the color property to set the background color to primary, secondary, or destructive. Also Buttons can have size as normal or large.


Attribute Description
color Color of the button theme. Can be primary, secondary, or destructive.
size Size of the button. Can be normal and large.

Medatus Buttons Group can be horizontal button groups set or vertical group set.


Attribute Description
alignment Alignment of the button group set can be horizontal and vertical.

Usage

Download the Medatus Library.


Add built Medatus Library to your project folder.


Add built Medatus Library Modules `med-button`, `med-button-group` to you packages list, with path to a library.


      npm install "file:`path the the button library module`"
      Example: 
      npm install "file:/medatusLibrary/med-button"
    
      npm install "file:`path the the button group library module`"
      Example:
      npm install "file:/medatusLibrary/med-button-group"

Add `MedButtonModule` or `MedButtonGroupModule` to you AppModule.

import { MatSlideToggleModule } from '@angular/material/slide-toggle';

@NgModule ({
  imports: [
    MatSlideToggleModule,
  ]
})

class AppModule {}

Add the <med-button> directive to the app.component.html element like so:

      <button med-button>Toggle me!</button>

Run your local dev server:

      ng serve

Then point your browser to http://localhost:4200

You should see the Medatus button component on the page.
